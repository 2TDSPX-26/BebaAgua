# BebaAgua

Este app foi criado para a matéria **Mobile Application Development** da turma **2TDSPX** do ano de **2026**.

O aplicativo tem como objetivo o controle da quantidade de copos de água ingeridos ao longo do dia.

Com este projeto, estamos aprendendo os passos iniciais do desenvolvimento mobile com **React Native** e **Expo**, além de alguns componentes principais, como:

- `View`
- `Text`
- `styles`
- `Flexbox`
